# Social Media Assets

Public image assets for scheduled X/Threads posts via Buffer.

Use folder format:

```text
YYYY/MM/slug.png
```

Raw URL pattern:

```text
https://raw.githubusercontent.com/novskidev/social-media-assets/main/YYYY/MM/slug.png
```

Only upload media that is safe to be public. Do not upload private photos, documents, tokens, screenshots with secrets, or sensitive metadata.
